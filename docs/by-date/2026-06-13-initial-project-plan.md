# 2026-06-13 Initial Project Plan

This date entry records the initial product direction and links to the canonical feature plan.

## Canonical Plan

## Decisions Captured

- Windows GUI application.
- Offline and self-contained portable-folder delivery.
- Bundled pretrained model weights are acceptable.
- First version exports `vocals`, `drums`, `bass`, `other`, `lowest`, and `highest`.
- Online videos are handled by system-audio recording during playback.
- CPU is required; compatible GPU acceleration is optional.
